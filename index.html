🐍 Install Python

  1. Download Python using this Link:
    https://www.python.org/ftp/python/3.14.4/python-3.14.4-amd64.exe
  2. During installation, tick the check-boxes for:
    • ✅ Admin Access
    • ✅ Add Python to PATH
    (Both appear on the 1st screen pop-up of the Python installation window.)
  3. To check your version & libraries, open Command Prompt (press Windows + R, then type cmd).
  4. Then paste the command below: 
    python --version && pip list

📦 Install Libraries

⚠️ Common Types of Errors & Resolutions

Note: Libraries built for newer Python versions may not be compatible with older Python environments. However, libraries developed for older versions generally remain compatible with newer Python versions, subject to deprecation risks.

To diagnose errors effectively, it is advisable to execute the program using the native Python IDE (IDLE) or terminal, as it provides clearer and more granular error traces.

Error Type Cause Resolution
Module Not Found Error Required library is not installed in the environment Install using pip install package_name
Import Error / Version Conflict Incompatible or incorrect library version Upgrade/install a compatible version using the single upgrade pip command (see above)
File Not Found Error File path is incorrect or file does not exist Verify file location, spelling, and ensure correct relative/absolute path is used
Permission Error File is open in another application or access is restricted Close the file, check permissions, or run the program with appropriate access rights
Indentation Error Improper indentation or inconsistent spacing Copy-paste code from AI platform as-is into the Python file; do not accidentally press any key that disturbs indentation
Key Error Referencing a column or key that does not exist Check column names using df.columns and ensure exact match (case-sensitive)
Value Error Incorrect data format or invalid value passed to a function Validate and clean input data before processing
Type Error Operation performed on incompatible data types Check and convert data types appropriately before performing operations
Authentication Error (API Key Issue) Missing or invalid API key for external services Set correct API key in environment variables or configuration file
